# fastapiproj1
     Описание
 Проект созданный для упрощение поиска нужных комплектующих при самостоятельном сборе Персонального Компьютера на базе ИИ(Oylan API).

  Требует ввода города, бюджета, характеристик предыдущего ПК,
   Желаемые требования (определенный ФПС в играх, процентный прирост производительности по сравнению со старой сборкой (При ее наличии.)).
  Выбора состояния комплектующих (Новое (запакованное), новое(открытое), Б/У).
  
  ИИ анализирует все предложения на рынке в выбранном городе (Через подключенный API), автоматически подбирает комплектующие, автоматически сравнивает их совместимость и определяет цену и состояние.

     Запуск
 1: Для запуска требуется открыть терминал в папке прокета и выполнить комманду
 uvicorn main:app --reload

 2: Перейти на сайт http://127.0.0.1:8000/

     Функции
 1: http://127.0.0.1:8000/ отображает статус работы ИИ асистента.

 2: http://127.0.0.1:8000/health проверка статуса работы системы.

 3: http://127.0.0.1:8000/docs сам интерфейс FastApi с интеграцией POST/Chat

 3.1: Healtcheck/Статус работы ассистента проверяется прямо с этой страницы при выборе нужной функции и нажатии Try it out>Execute.

 3.2: Запрос напрямую вводится в POST/Chat после нажатия Try it out
 
 3.3: Send message отправка сообщения/промпта
 
 3.4: GET/user_location Определение местоположения пользователя.

 3.5: POST/assistant/pc-config Отправка текущих характеристик ПК
